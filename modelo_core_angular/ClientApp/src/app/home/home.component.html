Modelo Sefaz de aplicação Asp Net Core com Angular

Modelo de aplicação Angular com uso dos padrões estabelecidos no Manual de Identidade do Governo do Estado de São Paulo.

Esse template inclui:
  • Folha de estilos Sefaz.css
  • Imagens
  • Menus:
    • Vertical - para acesso a recursos da aplicação
    • Popup - para informações e configuração da aplicação
  • Cabeçalho e rodapé nos padrões da Sefaz
  • Breadcrumb - barra hierárquica de links da página aberta
